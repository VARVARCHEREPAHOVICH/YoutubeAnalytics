📊 YouTube Analytics Titan — Clean UI Edition (C++)

YouTube Analytics Titan — это невероятно быстрое, стильное и полностью независимое десктопное приложение для глубокой аналитики YouTube-каналов.

Написано на чистом C++ (Win32 API) с кастомным графическим движком на базе GDI+. Никаких тяжеловесных фреймворков вроде Electron или Qt! Приложение весит всего пару мегабайт, работает молниеносно и не требует установки дополнительных библиотек.

✨ Главные особенности (Features)

⚡ Нативный рендеринг (GDI+): Полностью кастомный интерфейс с закругленными углами, плавными тенями и собственной физикой скроллинга.

🌌 Ультра-плавные анимации: Каскадное появление карточек видео (slide-in), плавная подсветка при наведении (hover) и многослойный пульсирующий неоновый спиннер загрузки.

🔄 Асинхронный многопоточный движок: Программа мгновенно отображает первые 50 роликов, продолжая "под капотом" скачивать до 500 видео с канала вместе с обложками. Интерфейс никогда не зависает!

🖼 Загрузка обложек в реальном времени: Бинарное скачивание превью (Thumbnails) напрямую с серверов YouTube (i.ytimg.com) через WinHTTP с безопасным клонированием в памяти GDI+.

🤖 Встроенный ИИ-Ассистент: Интерактивное всплывающее окно-инспектор для каждого видео с локальным эвристическим ботом, анализирующим показатели вовлеченности (ER), просмотры и лайки.

🌍 Полная поддержка Unicode: Поиск каналов на любом языке и с любыми спецсимволами.

📦 Portable-архитектура: Благодаря статической линковке (/MT), приложению не нужны сторонние DLL (например, VCRUNTIME140.dll). Скачал и запустил!

🛠 Инструкция по сборке (Visual Studio)

Проект легко собирается в Microsoft Visual Studio.

Скачайте YoutubeAnalytics.cpp

Откройте пустой проект C++ в Visual Studio и добавьте файл TitanApp.cpp.

Убедитесь, что в конфигурации выбрано Release и x64.

В настройках проекта (Свойства проекта -> C/C++ -> Создание кода) установите параметр Библиотека времени выполнения на Многопоточная (/MT) для статической линковки.

Нажмите Сборка -> Пересобрать решение.

Готовый независимый .exe файл появится в папке x64/Release.

Примечание: В коде используется мой API-ключ YouTube Data v3. Для использования в масштабных проектах рекомендуется заменить константу API_KEY на ваш собственный ключ из Google Cloud Console.

🚀 Как использовать

Запустите YoutubeAnalytics.exe. Вы услышите приятное гармоничное аудио-приветствие.

В левой панели введите handle нужного YouTube канала (например: @PewDiePie, @MrBeast или любой другой). Символ @ можно не писать, программа добавит его автоматически.

Нажмите Enter на клавиатуре.

Наслаждайтесь плавной анимацией загрузки и анализируйте статистику! Для детального анализа конкретного ролика просто кликните по его карточке.
